sinxra - инструкция по установке и использованию

1. Установка Radmin VPN:
- Скачать и установить Radmin VPN
- Создать сеть с друзьями
- Все участники подключаются к одной сети

2. Установка Python 3.11+:
- Скачать с python.org/downloads
- В установщике отметить "Add Python to PATH"
- Выбрать стандартную установку

3. Установка библиотек:
- Открыть командную строку (Win+R → cmd)
- Выполнить: pip install pynput pywinusb

4. Запуск программы:
- Один человек запускает сервер:  server.py
- Остальные запускают клиент:  client.py
- Ввести IP сервера (из Radmin VPN)

Техническая информация:
- Поддерживаемые клавиши: пробел, Enter, Tab, Esc, ЛКМ
